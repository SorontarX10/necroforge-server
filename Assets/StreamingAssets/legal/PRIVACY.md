# Privacy Policy (Steam Demo)

Last update: 2026-03-08

This demo processes leaderboard-related data such as `player_id`, `display_name`, run metrics, and anti-abuse validation metadata.

Production demo builds run with telemetry mode `OFF` and do not create local dev telemetry/perf/hitch files.

Data is used for leaderboard operation, integrity checks, and service reliability.
